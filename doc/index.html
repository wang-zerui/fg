华为云函数计算（FG）组件

组件说明

华为云函数组件是一个用于支持华为云函数应用生性周期的工具，基于Serverless Devs进行开发，通过配置资源配置文件s.yaml，您可以简单快速地部署应用到华为云函数计算平台。

本组件使用YAML规范（s.yaml）定义Serverless资源，定义了您的应用的函数、触发器等资源的配置。

快速开始

 
$ git clone https://github.com/xinwuyun/fg
$ cd cfc/example
$ s deploy
